# Pokemon Name Translator

Translates Pokemon, move, and item names between English, German, French, Italian, and Spanish.

**Live:** https://nyocon.github.io/pokemon-translator/

## Features

- Search by name in any of the 5 supported languages
- Tabs for Pokemon, Moves, Items, and Retro Mail words (experimental)
- Click any cell to copy the name
- Data is fetched once and cached in localStorage

## Data Sources

- Pokemon/Move/Item names: [PokéAPI](https://pokeapi.co/)
- Retro Mail word list: [Bl4ckSh4rk](https://gist.github.com/Bl4ckSh4rk/cf1e1aab540964bbeb8097404393b24d)
